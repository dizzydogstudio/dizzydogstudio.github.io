**应用名称：** Node Expense: Track Together
**平台：** iOS
**最后更新日期：** 2026-03-17

## 1. 接受条款
通过下载、安装或使用 Node Expense: Track Together，您同意受本服务条款的约束。如果您不同意，请勿使用本应用。

## 2. 服务说明
Node Expense: Track Together 是一款支出追踪应用，允许您：

- 追踪个人支出
- 与其他成员实时追踪并分摊群组支出
- 可选在支出中记录您的位置
- 将支出与选定的 iOS 日历同步
- 备份并可选加密您的数据（专业版订阅功能）
- 以 JSON 格式导出和导入支出数据（专业版订阅功能）

群组相关数据会同步到 Firebase，以便所有群组成员都能实时查看相同信息。除非您选择导出或备份，否则个人支出数据将保留在您的设备上。

## 3. 免费与付费功能

### 免费层级
- 每月 15 个个人支出名额
- 每月通过观看激励广告可额外获得最多 5 个名额
- 过去 7 天的手动日历同步

### 免费试用
- 14 天免费试用，可完整访问群组支出、数据保留和数据迁移功能
- 试用权限按 Apple ID 追踪，到期后无法重新开始

### 专业版（PRO）订阅
专业版订阅者可无限次使用：
- 无限个人支出追踪（无每月限制）
- 群组支出——创建和加入群组、分摊账单、记录结算
- 数据保留——带有可选 AES-256-GCM 加密备份的自动保留政策
- 数据迁移——以 JSON 格式导出和导入数据
- 无横幅广告、激励广告或插屏广告

除非在当前计费周期结束前至少 24 小时通过 Apple App Store 设置取消，否则订阅将自动续订。

### 终身订阅
"终身"通行证授予在开发者积极支持和分发应用期间访问专业版功能的权限。这并不构成应用或其云同步服务（如 Firebase、RevenueCat）将无限期可用的保证。如果应用停止开发、从 App Store 移除或开发者停止运营，则"终身"期限结束。

**退款与最终性：** 终身购买属于一次性付款。如果应用停止开发，或由于平台更改导致某些第三方集成（如 iOS 日历同步或 Firebase）不可用，将不提供按比例退款。

## 4. 群组功能与删除
群组支出通过 Firebase 在所有成员的设备上实时同步。参与群组即表示您理解：

- 您的显示名称和支出数据对群组所有成员可见
- 任何成员都可以向群组添加支出（受群组管理员设置的权限约束）
- 删除群组需要在 7 天投票期内获得所有成员的一致同意
- 超过 3 个月未活动的群组，如果提交了删除申请且在 7 天内没有成员拒绝，可能会被自动删除
- 群组管理员可以随时移除成员或转移数据
- 您在群组删除后选择保留的支出将保留在您的设备本地，并标记为来自已删除群组

应用的免费层级可能会显示：

- 插屏广告——当您达到每月支出限制时显示（每个应用会话一次）

激励广告的观看始终是可选的。插屏广告可能会在出现观看激励广告或订阅专业版的选项之前显示。广告提供商可能会按照我们的隐私政策所述收集匿名标识符。
