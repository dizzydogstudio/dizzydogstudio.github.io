# 隱私權政策
**應用程式名稱：** Node Expense: Track Together
**平台：** iOS
**最後更新：** 2026-03-17

## 1. 概覽
Node Expense: Track Together（以下簡稱「我們」）尊重您的隱私。本隱私權政策說明 Node Expense 所收集的資料、其使用方式，以及您在使用 Node Expense 時所擁有的選擇。

## 2. 我們收集的資料

### 2.1 您提供的資訊
- 費用記錄（金額、備註、類別）
- 群組資料（群組名稱、成員、設定）
- 群組費用與結算
- 顯示名稱（用於在共享群組中識別您的身份）
- 選填的位置資料（僅在您啟用時收集）
- 行事曆存取（僅限您所選擇的行事曆）

### 2.2 自動收集的資料
- 匿名使用分析（透過 Firebase Analytics）
- 匿名裝置識別碼 — 用於在重新安裝後維持您的訂閱及應用程式資料
- 當機記錄與診斷資訊
- 裝置與應用程式效能資訊
- 推播通知權杖 — 在啟用時用於傳送群組活動更新及個人化閾值提醒

此資料僅用於改善 Node Expense 的穩定性、效能及使用者體驗。

### 2.3 訂閱與付款
購買及訂閱由 Apple 與 RevenueCat 處理。我們不會收到您的信用卡或付款詳細資訊。RevenueCat 儲存匿名訂閱者識別碼，以管理您的訂閱狀態、「終身」使用權及免費試用期。此資料的保存期限以驗證您在重新安裝後存取 PRO 功能所需為限，並與您的群組費用雲端資料分開處理。

## 3. 第三方服務
Node Expense 使用以下第三方服務：

- RevenueCat — 訂閱與應用程式內購管理，包括免費試用追蹤
- Google AdMob — 橫幅、獎勵及插頁式廣告
- Firebase Analytics — 匿名使用分析
- Firebase Firestore — 群組資料同步（費用、成員、結算、刪除請求）
- Firebase Authentication — 用於訂閱連續性的匿名使用者識別
- Firebase Cloud Functions — 背景工作，包括群組刪除及推播通知傳送
- Firebase Cloud Messaging（FCM）— 群組活動的推播通知

Firebase 僅作為群組相關功能的同步層。個人（非群組）費用資料不會上傳至雲端服務。

## 4. 廣告
免費方案的使用者將在 Node Expense 中看到廣告支援。這些廣告可能包含橫幅廣告、達到每月上限時的插頁式廣告，或用於解鎖額外費用記錄槽位的選填獎勵廣告。

為提供更個人化的體驗，我們可能會請求您的許可來追蹤您的活動。這有助於我們的廣告合作夥伴投放更相關的廣告並衡量其成效。若您選擇不被追蹤，您仍會看到相同數量的廣告，但廣告的相關性可能較低。您的財務資料將保持私密，絕不會與廣告商共享。

## 5. 位置資料
位置資料為選填，僅在您在「設定」中明確啟用以進行費用追蹤時才會收集。啟用後，您的位置將隨新費用一同記錄。位置資料儲存於您的裝置本機，不會上傳至雲端服務。您可以隨時在 iOS 設定中停用位置存取。

## 6. 行事曆存取
Node Expense 僅存取您所選擇的行事曆。行事曆存取僅用於以行事曆事件的形式顯示及同步費用項目。行事曆資料絕不會與第三方共享。

## 7. 資料儲存、保留與加密

### 7.1 裝置本機儲存
以下資料使用 SwiftData（Apple 的安全本機儲存框架）儲存於您的裝置本機：
- 個人（非群組）費用記錄
- 應用程式設定與偏好設定
- 群組資料的本機副本（供離線存取）
- 群組費用的快取（供離線檢視）

個人（非群組）費用資料不會離開您的裝置，除非您明確匯出或備份。已刪除群組的費用將在您的裝置本機保留，並標示為「已刪除群組」，以確保您的記錄得到保存。

### 7.2 群組功能的雲端儲存
針對群組功能，所選資料將同步至 Firebase Firestore，讓所有群組成員能夠查看相同資訊：

- 群組文件（名稱、說明、圖示、成員、權限、分享碼）
- 群組費用（金額、類別、備註、日期、位置（若已啟用）、付款人身份）
- 群組結算（成員之間的付款記錄）
- 群組刪除請求及成員投票
- 試用開始日期（用於重新安裝後的復原）
- 推播通知權杖（FCM）

雲端同步僅用於群組功能。個人費用絕不會被上傳。

### 7.3 加密與備份
- 具有功能存取權限的使用者（PRO 訂閱或有效免費試用）可啟用使用 AES-256-GCM 加密的加密備份。
- 您可控制費用記錄的保留時間（全部時間或指定年數）。
- 備份檔案會匯出至您所選擇的位置（iCloud 雲碟、本機裝置或其他 iOS 檔案位置）。
- 備份加密金鑰由系統自動管理，並僅儲存於您的 iCloud 鑰匙圈中，絕不會傳輸至我們的伺服器，也不會被我們開發者存取。此金鑰不會顯示在「密碼」應用程式中，但會透過 iCloud 安全同步至您的其他裝置。只有登入您 Apple ID 的裝置上的應用程式，才能存取此金鑰以加密或還原備份。

## 8. 資料遷移
具有功能存取權限的使用者（PRO 訂閱或有效免費試用）可以 JSON 格式匯出及匯入其資料。Node Expense 無法存取您的匯出檔案。匯出檔案會儲存至您所選擇的位置（iCloud 雲碟、本機裝置或其他 iOS 檔案位置）。

## 9. 資料刪除
您可以透過以下方式刪除您的資料：
- 在 Node Expense 中刪除個別或全部個人費用
- 離開或請求刪除群組 — 當所有成員同意後，群組資料將從 Firebase 移除
- 標示為「刪除時保留」的群組費用將保留於您的裝置本機，並標記為已刪除群組的資料；這些資料不會被重新上傳
- 解除安裝 Node Expense 將移除您裝置上所有本機儲存的資料

注意：Firebase 群組資料（費用、結算、成員記錄）可能在群組透過應用程式內的刪除流程正式刪除之前持續存在。

## 10. 兒童隱私
Node Expense 不適用於 13 歲以下兒童。我們不會在知情的情況下收集 13 歲以下兒童的個人資料。若您認為兒童已提供個人資訊，請與我們聯絡。

## 11. 您的權利
您可以：
- 在 iOS 設定中停用位置存取
- 在 iOS 設定中撤銷行事曆存取
- 在 Node Expense 中刪除您的本機費用資料
- 在刪除前匯出您的資料
- 請求刪除群組（將為所有成員從 Firebase 移除群組資料）
- 隨時透過 Apple App Store 設定取消您的訂閱
- 隨時停止使用備份或加密匯出功能

## 12. 本政策的變更
我們可能會不定期更新本隱私權政策。變更將反映於本文件頂部的「最後更新」日期。在變更後繼續使用 Node Expense，即表示您接受更新後的政策。
